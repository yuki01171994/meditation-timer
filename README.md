# Serein

A quiet place between you and the world.

## What it is
Serein is a minimal meditation timer for iPhone.
No goals. No streaks. No coaching.
Just time you can return to — and a history that quietly accumulates.

## Core features (v1)
- Timer with minute/second wheel pickers
  - Seconds: 5-second step by default
  - Optional 1-second step in Settings
- Completion notification
  - Sound / Haptics / Both / None
- Gentle completion hold (00:00 stays for 3 seconds)
- History with Apple Watch–inspired rings
  - Day / Week (Monday start) / Month / All-time
- Sessions are stored locally
  - Completed and interrupted sessions are both recorded

## Not included
- Breathing guide
- Post-meditation notes
- Goals, streaks, badges, rankings
- Reminders

## Tech (planned)
- iOS + SwiftUI
- SwiftData for local storage
